<h1 class="textMTitle" itemprop="headline">Женский масс-старт на этапе КМ по биатлону в Ханты-Мансийске отложен из-за ветра</h1>

						<p>Москва. 20 марта. INTERFAX.RU - Женский масс-старт на этапе Кубка мира по биатлону в Ханты-Мансийске перенесен на 18:45 по местному времени (16:45 по Москве) из-за сильного ветра, сообщили журналистам организаторы соревнований.</p>

<p>"Из-за сильного ветра падают деревья, ветки, и есть опасность для спортсменов на трассе", - пояснили в оргкомитете.</p>

<p>Женский масс-старт должен был начаться в 14:20 по местному времени (12:20 по московскому времени).</p>

<p>Время начала мужского масс-старта осталось прежним - 17:00 по местному времени (15:00 по Москве).</p>

<p>Окончательное решение о судьбе воскресных гонок будет принято в 16:00 по местному времени (14:00 по Москве), сообщили организаторы соревнований.
